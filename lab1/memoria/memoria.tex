\section{Introducción}

Detección automática de contenido sexista en tweets usando el dataset EXIST 2025. Clasificación binaria (sexista/no sexista) mediante modelos clásicos y modelos de lenguaje con fine-tuning.

\section{Metodología}

\subsection{Preprocesamiento}

\begin{itemize}
    \item V1: tweet original + text\_clean básico (lowercase, elimina URLs/menciones/hashtags).

    \item V2: Mejoras en text\_clean: eliminación de stopwords, preservación de negaciones \{no, nunca, jamás, nada, nadie, tampoco, ni, sin\} con marcado NEG\_, normalización de acentos/elongaciones.
\end{itemize}

\subsection{Modelos evaluados}
\begin{itemize}
    \item Clásicos (TF-IDF): Logistic Regression, LinearSVC, Random Forest, Gradient Boosting, Naive Bayes, Stacking.
    \item Modelos de embeddings con LoRA: F2LLM-4B y KaLM.
    \item LLMs con LoRA: Ministral-3-8B-Instruct.
\end{itemize}

\section{Conclusiones}

\begin{itemize}
    \item \textbf{Modelos no clásicos}: Superan modelos clásicos +0.08 puntos aproximadamente en F1-Macro (F2LLM-4B: 0.8593 vs Stacking V2: 0.7801).
    \item \textbf{Clásicos}: Stacking empeora en F1-Macro V2 (-0.01). LogReg mejora en V2 (+0.05 puntos).
    \item \textbf{Preprocesamiento}: Tweet original es mejor que el texto limpio en general.
    \item \textbf{Ensemble top 5}: Idéntico a F2LLM-4B individual. No aporta mejora.
\end{itemize}
